Klicke auf eine Karte um sie zu spielen, oder auf das Deck um eine zu ziehen.

Eine Karte kann gespielt werden, falls sie die gleiche Zahl oder Farbe hat.


Bei "+2" und "+4" muss der Gegner zusätzliche Karten ziehen.

Diese Sonderkarten können immer gelegt werden.


Wenn das Deck leer ist, wird der Ablagestapel als neues Deck gemischt.

Der Spieler der zuerst keine Karten auf der Hand hat, gewinnt das Spiel!
